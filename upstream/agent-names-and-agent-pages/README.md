# Upstream reference: Agent Names and Agent Pages

This folder keeps **local reference notes** used to derive and justify controls in this repository. The authoritative upstream work is the ToIP Agent Names Task Force repository: `https://github.com/trustoverip/dtgwg-agent-names-tf`.

## Files

- `draft-2026-02-07-v5-extract.md` — non-normative historical working extract kept for local traceability only.

## How it is used

- The normative baseline (`spec/agent-name-assurance-baseline.md`) maps key upstream concepts to:
  - ANAGB controls
  - evidence expectations by tier (AN-0…AN-3) and assurance level (AL1–AL4)

This repo does **not** claim conformance to the upstream repository; it uses that work as a **design input** and converts it into controls, conformance artifacts, and evidence expectations.
