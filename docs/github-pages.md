# GitHub Pages

This repository can be published with **GitHub Pages** from the `/docs` folder.

## Recommended configuration

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

## Entry points

- `docs/index.md` (this file) is the recommended landing page.
- The normative spec remains under `/spec` and is linked from the docs landing page.


- `docs/ais1-interpretation-profile.md` is the GitHub Pages landing page for AIS-1-related relying-party guidance.
