# v0.4.0

This release introduces **transport alignment** guidance with the ToIP **Trust Spanning Protocol (TSP)** for agent identifier trust tasks and improves repo operational reliability.

## Highlights

- Added transport alignment documentation for TSP and clarified where transport shows up in identifier verification tasks.
- Updated architecture snapshot to include the transport layer and fixed Mermaid syntax for GitHub rendering.
- Added Dependabot configuration for GitHub Actions and Python dependency updates.
- Hardened GitHub Actions workflows (concurrency, permissions, caching) and added Mermaid parsing checks to prevent doc regressions.

## Notes

- ANAB remains identifier- and transport-agnostic; TSP is referenced as a candidate transport substrate.
