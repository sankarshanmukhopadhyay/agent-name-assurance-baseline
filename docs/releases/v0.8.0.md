# v0.8.0

This release adds an **experimental runtime identity lifecycle layer** to ANAB.

## Highlights

- Added `docs/runtime-identity-lifecycle.md` for higher-assurance, runtime-governed named agents
- Added `docs/experimental/agent-governance-toolkit-alignment.md` to document the upstream conceptual adoption boundary
- Refreshed assurance-level and implementer guidance so runtime identity expectations are visible for AL3 and AL4 without altering the normative baseline

## Notes

- The ANAB control catalog, declaration schema, and evidence-bundle schema remain unchanged.
- The Microsoft Agent Governance Toolkit is used only as an upstream conceptual reference for experimental runtime guidance.
