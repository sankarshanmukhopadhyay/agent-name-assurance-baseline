# v0.5.1 — Align ANAB to OTAM and trust-infrastructure-schemas

## Changed

- Updated ecosystem references from `schemas` to `trust-infrastructure-schemas`.
- Clarified that the upstream schema repository now implements the Open Trust Artifact Model (OTAM).
- Updated architecture, interoperability, and transport documentation so ANAB is positioned as a domain baseline layered on canonical trust artifact schemas.

## Compatibility

This is a documentation and positioning release only. No ANAB control semantics, JSON schema bodies, or conformance requirements changed.
