# v0.5.0

This release makes ANAB materially more useful for **A2A-exposed named agents**.

## Highlights

- Added explicit A2A alignment guidance in `docs/a2a-alignment.md`
- Added **Annex K** for A2A concept-to-control mapping
- Expanded the normative control catalog with six A2A-facing controls covering:
  - signed Agent Cards
  - page/card/declaration consistency
  - extended metadata gating
  - task/context/tenant scoping
  - authenticated streaming and callbacks
  - safe media-type declaration and handling
- Updated the applicability matrix, checklist, and threat model so assessors can actually use the repo in real A2A deployments instead of waving it around like a ceremonial PDF

## Notes

- ANAB still does **not** claim that name assurance alone proves operational safety or authority.
- For endpoint-level assurance, pair this release with DTG Conformance Assurance v0.5.0.
